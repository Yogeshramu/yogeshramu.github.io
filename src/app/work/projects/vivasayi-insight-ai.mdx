## Overview

Engineered an AI-driven agricultural platform tailored for localized farmers in English and Tamil. This intelligent application leverages multimodal AI models from Groq (Llama 3 and Vision) to predict soil moisture, detect crop diseases through visual inference, and provide localized weather-based advisories.

## Key Features

- **Computer Vision Integration**: Utilizing Llama-4-Scout models, parsing uploaded leaf or crop images to reliably diagnose potential plant pathology defects constraints.
- **Chatbot Artificial Intelligence**: Multilingual AI conversational agent driven by Groq Cloud APIs to instantly answer context-specific agricultural questions.
- **Predictive Moisture Analysis**: Machine learning correlation utilizing live environmental metrics (Temp, Rainfall, Humidity) to proactively map field irrigation needs.
- **Weather API Integrations**: Syncs tightly with OpenWeatherMap APIs to push threshold alerts representing frost risk or severe winds to the user interface.
- **Security & User Isolation**: Full authentication and history persistence through NextAuth.js combined with PostgreSQL and Prisma ORM safely securing all API transactions.

## Technologies Used

- **Next.js 14 / React**: React application framework driving frontend and API routing logic
- **Groq AI (Llama 3 / Vision)**: AI/ML processing core responding to text/image prompts
- **PostgreSQL / Prisma**: High-reliability schema mapping and data queries
- **NextAuth.js**: Standardized credentials tracking
- **OpenWeatherMap**: Environmental data streams

## Outcome

Constructed a sophisticated, deeply integrated AI application that shifts heavy ML computational workloads off user devices utilizing serverless API patterns. Demonstrates proficiency in handling 3rd-party microservices (Auth, DB, ML, Weather APIs) and centralizing them into a cohesive, user-centric next-generation platform.
