## Overview

Designed and implemented a production-grade AWS architecture for deploying containerized backend services at scale. The system leverages AWS ECS (Fargate) for serverless container orchestration, removing the need to manage underlying EC2 instances while delivering high availability across multiple availability zones.

## Key Features

- **Secure VPC Architecture**: Designed a multi-tier AWS VPC with public and private subnets across multiple availability zones, enabling high availability and network isolation for containerized applications.
- **Containerized Deployments**: Containerized backend services using Docker with optimized multi-stage builds. Images are stored and versioned in Amazon ECR and pulled directly by ECS task definitions.
- **AWS ECS (Fargate)**: Deployed services on ECS Fargate, eliminating server management overhead while enabling auto-scaling based on load metrics.
- **Application Load Balancer**: Configured ALB with target group health checks to route traffic to healthy ECS tasks and support zero-downtime rolling deployments.
- **Managed Database**: Integrated AWS RDS in private subnets with appropriate security group rules to securely connect to application containers.
- **GitLab CI/CD Pipeline**: Automated the full delivery lifecycle — Docker image build → push to ECR → trigger ECS rolling deployment — on every code merge.

## Technologies Used

- **AWS ECS (Fargate)**: Serverless container orchestration
- **Docker**: Application containerization
- **Amazon ECR**: Container image registry
- **AWS RDS**: Managed relational database
- **Application Load Balancer**: Traffic management and health checks
- **VPC**: Network isolation with public/private subnets
- **GitLab CI/CD**: Automated build and deployment pipeline

## Outcome

Achieved a fully automated, secure, and highly available container deployment environment. Rolling deployments via GitLab CI/CD eliminated manual steps and reduced deployment risk, while multi-AZ architecture ensures fault tolerance for production workloads.
